The goal of this project is to simulate real-world Data Analyst tasks by answering business questions through SQL queries, documenting the analysis process, and demonstrating best practices for relational databases.

# 📖 About

This repository documents my journey exploring the **DVD Rental** database using **PostgreSQL**.

Rather than simply solving SQL exercises, this project focuses on developing analytical thinking by investigating business scenarios, exploring relational data, and producing insights through SQL.

# 📊 Business Questions

This project aims to answer questions such as:

- Which customers rent the most movies?
- Which movie categories generate the highest revenue?
- Which actors appear in the most films?
- Which movies are rented most frequently?
- Which store generates the highest revenue?
- Which staff member processed the most rentals?
- What is the average rental duration?
- Which customers spend the most money?

More questions will be added throughout the project.
